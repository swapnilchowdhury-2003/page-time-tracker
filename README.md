# Page Time Tracker 🕐

A simple Chrome extension that tracks how much time you spend on each website.

## Features
- Auto starts tracking when you visit any page
- Shows a live timer for the current site
- Ranked list of all sites with time spent
- Reset button to clear all data

## How to Install

1. Download this repository — click the green "Code" button → "Download ZIP"
2. Extract the ZIP file on your computer
3. Open Chrome and go to: chrome://extensions
4. Turn on "Developer mode" (toggle in the top-right corner)
5. Click "Load unpacked"
6. Select the folder named "page-time-tracker" (or whatever the extracted folder is called)
7. Done! The extension is now active and tracking your time

## How to Use

- Click the extension icon in your Chrome toolbar to see your stats
- If you don't see the icon, click the 🧩 puzzle piece in Chrome → pin "Page Time Tracker"
- Browse normally — it tracks automatically in the background
- Click "Reset All" inside the popup to clear all data and start fresh
